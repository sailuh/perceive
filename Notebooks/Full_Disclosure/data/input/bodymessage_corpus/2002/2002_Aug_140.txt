Security Advisory  Subject:                Linux: buffer overflow in multiple DNS resolver libraries Advisory number:        CSSA-2002-034.0 Issue date:             2002 August 05 Cross reference: ______________________________________________________________________________   1. Problem Description          From CERT CA-2002-19: A buffer overflow vulnerability exists in         multiple implementations of DNS resolver libraries. Operating         systems and applications that utilize vulnerable DNS resolver         libraries may be affected. A remote attacker who is able to         send malicious DNS responses could potentially exploit this         vulnerability to execute arbitrary code or cause a denial of         service on a vulnerable system.   2.
